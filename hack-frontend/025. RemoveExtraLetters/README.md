# Удаление лишнего дублирования букв

Напишите функцию, которая принимает строку, содержащую лишнее дублирование букв (например: 'нууужно'), и возвращает строку, где каждая буква встречается только один раз подряд.

- Регистр букв сохраняется.
- Символы, не являющиеся буквами, также остаются в строке, но без дублирования.

## Примеры:

```ts
Input 1: "нууужно"
Output 1: "нужно"
Input 2: "ллллиииишшшнийййй тттексттт"
Output 2: "лишний текст"
```
