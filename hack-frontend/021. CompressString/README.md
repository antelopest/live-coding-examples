# Сжатие строки

Напишите функцию compressString(str), которая принимает строку str и возвращает её сжатую версию. 

## Условия:

Сжатая версия строки должна содержать символы исходной строки и количество их повторений подряд.
Если символ встречается один раз, он включается в строку без числа.

## Примеры:

```
Input 1: "AAB"
Output 1: "A2B"
Input 2: "ABC"
Output 2: "ABC"
```
